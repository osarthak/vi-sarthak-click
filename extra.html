Installing apache server

https://www.redhat.com/sysadmin/install-apache-web-serverhttps://www.digitalocean.com/community/tutorials/how-to-install-the-apache-web-server-on-ubuntu-20-04

(what is apache)

Apache server is most widely used web server. it is easy to setup within minutes. apache is used to serve website on the web

(installing apache)

to install apache on ubuntu following command can be enter to install the package on the ubuntu server

sudo apt install apache2

apache will automatically setup and started it service, to see if the apache service is running on our host, type the ip address in webbrowser that is something like this http://192.168.1.10. if you install it correctly it will show default apache html website.

(apache configartion)

Using your text editor of choice such as vim or nano to edit /etc/httpd/conf/httpd.conf.

These two values may already be set in that file, but confirm them to be sure:

DocumentRoot /var/www/html
Listen 80

the default apche web page is located at /var/www/html. you can edit the html code as you desire like

<html>
<head>
<title>Welcome to my test site!</title>
</head>
<body>
This site is for testing purposes only
</body>
</html>

The Listen directive simply indicates which port the service listens on. In this case, use the standard port 80 for HTTP.

Using the Network File System

https://www.redhat.com/sysadmin/configure-nfs-linux

Network file system is a distributed file system protocol which means that we able to mount the remote host directory to the local host.

Here are the proper way to sue network file system

  1. Setup NFS on remote server: To install the NFS on Redhat Linux Distribution run the following command in terminal 
$ sudo dnf install nfs-utils

Start the service NFS Service

$ sudo systemctl enable --now nfs-server

Now its time to share folder which you want to share to client computer, for this instance /home/user/Downloads and followed by the subnet address of the local netowrk. To do that you have to add the line in /etc/exports use any editor such as nano or vim to add this line /home/user/Downloads 192.168.122.0/24(rw). after just type sudo exportfs -r

Set up your client: Setting up the nfs on client side is fairly easy all you have to do is type following command that is sudo mount - v -t nfs 192.168.122.10:/home/user/Downlaods /home/client/nfs-mount . in last command mount used to mount the new volume in our system that nfs followed by the ip address of a nfs-server and its shared folder and atlast where to mount the volume that is /home/client/nfs-mount
