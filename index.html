  3. Managing passwords     [cached]
    $ sudo passwd [username] [parameter]

    Here are some comman parameter used to manage passwords

    1. -S = See the STATUS of the user account password
    2. -l = used to LOCK the user password
    3. -u = to UNLOCK the user account password
    4. -d = to DELETE the password (user will login the account without password)
    5. -e = immedatly EXPIRE the password. force user to change the password by expiring user password
    6. -n = user won't able to change the password before the NUMBER given with -n
    7. -x = the MAX number of days remain valid after that user got promt to change the password
    8. -w = get WARN by receiving warnings that the password will expire.
    9. -i = account get expire if user INACTIVE or didn't change the password after the expire time
  4. Granting system Administrator privileges to regular user     [cached]

    Two way to grant Administration privileges

    1. by adding user in sudo or wheel group
      2. $ sudo usermod -a -G wheel [username]

      or

      $ sudo usermod -a -G sudo [username]
    2. by editing /etc/sudoers file and add line [username] ALL=(ALL) ALL
      3. $ sudo nano /etc/sudoers
